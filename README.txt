# Quantum Dig Bot
Фейковый Telegram-бот для майнинга GMC/XRP.

## Как запустить

1. Установи Python: https://www.python.org/downloads/
2. Установи зависимости:
    pip install python-telegram-bot==13.15
3. Вставь токен в config.py
4. Запусти бота:
    python bot.py

## Команды:
- /start — начало игры
- /mine — добыча XRP
- /balance — ваш баланс
- /invite — пригласить друга и получить GMC

Хранилище: SQLite (файл users.db создаётся автоматически)
